Logica condizionale

Qualunque script in un ultima analisi è una sequenza di istruzioni che vengono eseguite dall'engine javascript
Le nostre istruzioni (tutto quello che scriviamo all'intenro di uno script) sono un flusso di comandi
questi comandi stabiliscono il comportamento del sistema (nel mio caso la pagina web) per portarlo da uno stato iniziale a uno stato finale. In altre parole il programma ci permette di descrivere un algoritmo tramite cui possiamo raggiungere un certo obiettivo oppure un certo risultati.
Per ottenere un risultato avremo bisogno di alcune istruzioni che ci aiutano a stabilire il flusso di esecuzione del nostro programma.

Blocchi di codice

Più istruzioni possono essere raggruppate in blocchi che fungono come una singola istruzione. I blocchi di codice sono importanti quando evidenziamo che un gruppo di istruzioni deve essere eseguito interamente
tipicamente nella logica condizionale eseguiamo dei blocchi di codice in seguito alla valutazione di una condizione
un blocco di codice è racchiuso tra parentesi graffe

la logica condizionale è una categoria particolare di istruzioni che consentono di eseguire blocchi di codice alternativi in base alla verifica di una condizione. Lo script in un certo senso "decide" se eseguire un blocco di codice oppure un altro, la decisione sarà basata sulla verifica della condizione.
Javascript prevede due istruzioni condizionali

  • if
  • switch

If prevede tre forme

  • if semplice
  • if con alternativa (if..else)
  • if a cascata (if..else if..else)

If semplice

La forma if semplice esegue un blocco di codicesolo se una condizione è vera
se la condizione non è vera il blocco di codice non viene eseguito
