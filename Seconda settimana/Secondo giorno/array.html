Array

L'array è una variabile multidimensionale. Dentro una variabile posso inserire un solo valore, dentro a un array posso inserire più valori.

Dentro un array incontriamo strutture di dati omogenee

Dentro a un array possiamo conservare qualunque tipo di dato (primitivi o oggetti)

Gli array sono degli elenchi / delle liste dentro a cui salvare diversi tipi di dati

Dentro a un array posso associare più valori ad un unico nome di variabile.

L'uso degli array ci evita di definire più variabili e semplifica lo svolgimento delle operazioni

Proprietà degli array

In javascript esistono una serie di metodi che si indicano con nome_array.nome_metodo
che ci permette di fare una serie di operazioni sul mio array.

La prima proprietà che vediamo è length

La proprietà length conta quanti elementi sono contenuti in un arrya
