Metodi delle stringhe

Abbiamo già visto alcuni metodi: length (lunghezza), slice, substring, substr (tre metodi per tagliare delle stringhe), toUpperCase e toLowerCase (per fare stringhe maiuscole | minuscole), trim (pulire dagli spazi bianchi)

Metodi di controllo sulle stringhe introdotti da EcmaScript6: mi permettono di controllare se una stringa inizia, finsice o contiente determinati caratteri.

Controllo se una stringa inizia con determinati caratteri -> metodo startsWith

Controllo se una stringa finisce con determinati caratteri -> metodo endsWith

Controllo se una stringa contiene una determinata parola -> metodo includes
