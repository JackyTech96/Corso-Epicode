Tipi di dati

Javascript prevede cinque tipi di dati primitivi: numeri, stringhe, booleani, null e undefined
e un tipo di dato complesso: gli oggetti.

Qualsiasi altro elemento previsto dal linguaggio (array, funzioni ...) sono tipi di dati oggetto

Negli oggetti noi possiamo inserire proprietà (attributo/valore) e/o metodi (funzioni)

Tipo di dato string

è una sequenza di caratteri delimitata da apici doppi oppure singoli
"questa è una stringa" - 'questa è una stringa' -> non c'è regola per quale delimitatore usare
unica regola: delimitatore iniziale deve essere uguale al delimitatore finale "l'altro ieri" - "questa è una 'stringa'"
esiste anche la stringa vuota "" ''
Per inserire caratteri speciali in una stringa -> \
"Ieri c'era il sole.\nOggi piove" -> \n inserisce un ritorno a capo
'l\'altro ieri' - "questa è una \"stringa\"" -> il delimitatore preceduto da \ si considera come carattere normale

Tipo di dato numerico

Senza apici. Esiste un solo tipo di dato numerico per tutto (interi, decimali, negativi)

Javascript è un linguaggio a tipizzazione debole, non c'è controllo sul tipo di dato

Tipo di dato booleano

ha solo due valori possibili: true | false

Tipo di dato null

Prevede un solo valore possibile: null -> non rientra nei precedenti tipi di dato

Tipo di dato undefined

Prevede un solo valore: undefined. Rappresenta un valore che non esiste. Non ha assegnato nessun valore (nemmeno null)
