Operatori

Tramite gli operatori posso combinare valori, variabili tra di loro

Gli operatori determinano il risultato di un'operazione

Classificazione gli operatori in base al numero di valori che possono combinare: unari, binari, ternari

Operatori aritmetici

operatori binari perché combinano due valori (numerici!) + addizione - sottrazione / divisione * moltiplicazione % modulo o resto esempi: 9/4 x + 10 3 * 2 e seguono le regole delle precedenze matematiche come in matematica posso usare le parentesi() 4 + 5 * 6 + 7 = 41 (4 + 5) * (6 + 7) = 117

operatori aritmetici possono anche essere unari, cioè combinano solo un numero ++ incremento -- decremento - negazione -> x=5 -> -x = -5 -> incremento e decremento li posso applicare solo a delle variabili -> è molto importante la posizione dell'operatore-> se l'operatore è a sinistra dell'operando l'operazione viene eseguita prima di valutare l'espressione -> se l'operatore è a destra dell'operando l'operazione viene esguita dopo

Operatori logici

consentono di combinare espressioni booleane(true|false)

&& -> operatore binario -> and || -> operatore binario -> or ! -> operatore unario -> not 5 > 2 && 3 !=4 // true !5 === 5 // false = -> assegnamento -> x = 3 + 5; == -> uguale != -> diverso === -> strettamente uguale (uguale sia nel valore che nel tipo di dato) -> esempio: 5 e "5" sono uguali nel valore ma non nel tipo di dato -> non sono strettamente uguali

Operatore ternario

Permette di combinare tre dati tra di loro. Restituisce un valore in base a un'espressione booleana

Sintassi: condizione ? valore1 : valore2
Verifico se la condizione è vera (true|false) -> se è true viene restituito il primo valore altrimenti il secondo

Operatori relazionali

operatori utilizzati per fare un confronto tra valori

Operatore Nome
< minore
> maggiore
<= minore o uguale
>== maggiore o uguale
== uguale
!= diverso
=== strettamente uguale
!== strettamente diverso

Queste espressioni restituiscono un valore booleano (true|false)

Operatori su stringhe

Operatore concatenazione (+) -> crea una nuova stringa come risultato della concatenazione di due stringhe
