Inserire codice inline

Posso inserire direttamente le istruzioni javascript in un elemento HTML, assegnandolo a un attributo che rappresenta un evento

Altro approccio per inserire codice inline, ma è valido solo per i link

Inserisco blocchi di codice

Pro: nel caso abbiamo poco codice javascript metto in relazione subito il codice da eseguire con il suo tag Html

Contro: nel caso il codice sia minimamente complesso (variabile, funzioni, oggetti) non risulta comodo

Tag head o tag body? primo ordine di considerazione -> performance; secondo ordine di considerazione -> che il tag su cui js interagisce sia già stato parsato (letto e quindi disponibile) -> questo spiega perché tante volte troviamo gli script in fondo alla pagina prima di </body>

Inserisco javascript esterno

Approccio più consigliato, Consiste nel collegare a una pagina html uno script presente in un file esterno
Il grande vantaggio è la separazione tra struttura e la programmazione, tra cos'è un tag e cosa deve fare

Oggetto console

La console è uno strumento fondamentale per scrivere codice javascript (per qualsiasi linguaggio di programmazione)

La console ci permette di fare il debug, cioè ci permette di controllare il flusso del nostro codice

Spesso la console si usa per tenere traccia dell'evoluzione del programma o verificare dei valori

Per vedere la console tasto destro -> ispezione -> console
